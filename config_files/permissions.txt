You can use SimpleSpleef with built-in permission plugin. The following rights are available:

simplespleef.list List command
simplespleef.play Join dog-eats-dog command.
simplespleef.team Join team commands.
simplespleef.leave Leave command.
simplespleef.start Start game.
simplespleef.stop Stop game.
simplespleef.delete Delete game.
simplespleef.reload Reload configuration file.
simplespleef.prize Use the prize command to set a prize for the next winner (money or item id).
simplespleef.startsingle Start the game as a single player.
simplespleef.admin Administration commands
